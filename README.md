# Maryam-Asda-Password-Strength-Analyzer# Password Strength & Email Validator
A Python tool to verify email formats and analyze password security.

## Key Features
- **Email Validation:** Uses Regex to ensure proper email formatting.
- **Security Check:** Evaluates passwords based on length and complexity.
- **Instant Feedback:** Helps users create stronger credentials.

## Technology
- **Language:** Python 3.x
